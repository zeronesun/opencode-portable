# 故障排除指南

## 常见问题

### 1. 无法在旧系统上运行
**症状**：运行时出现 `GLIBC_2.XX not found` 错误
**解决方案**：本项目已经使用musl静态编译，不需要系统GLIBC。如果仍然出现此错误，请确保你下载的是正确的架构版本。

### 2. 中文显示或粘贴问题
**症状**：中文显示乱码、宽度计算错误、粘贴内容混入`[Pasted N lines]`
**解决方案**：本项目已经修复了#17032号CJK粘贴bug。如果仍然有问题，请提交issue并提供你的终端模拟器名称和版本。

### 3. 调用系统命令出错
**症状**：运行`git`、`ssh`、`curl`等系统命令时出现奇怪的错误
**解决方案**：本项目使用了LD_PRELOAD清理器，应该已经解决了这个问题。如果仍然有问题，请尝试在命令前加上`env -u LD_PRELOAD`。

### 4. 版本号显示错误
**症状**：运行`opencode --version`显示`unknown`
**解决方案**：这是构建过程中的问题，请等待下一个自动构建版本，或者手动触发一次构建。

## 支持的系统和架构

### 支持的架构
- x86_64 (amd64)
- arm64 (aarch64)

### 测试过的系统
- CentOS 7 (GLIBC 2.17)
- Ubuntu 20.04 (GLIBC 2.31)
- QNAP TS-453D
- Synology DS920+
- Raspberry Pi 4/5
